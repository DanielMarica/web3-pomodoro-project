# Fichiers audio pour le Pomodoro Timer

Ce dossier contient les fichiers audio utilisés pour les notifications :

- **work-complete.mp3** : Son joué quand un pomodoro de travail est terminé
- **break-complete.mp3** : Son joué quand une pause est terminée  
- **tick.mp3** : Son de tic-tac (optionnel)

## Comment ajouter les sons

1. Télécharge des sons gratuits depuis :
   - [Mixkit](https://mixkit.co/free-sound-effects/)
   - [Freesound](https://freesound.org/)
   - [Zapsplat](https://www.zapsplat.com/)

2. Renomme-les et place-les dans ce dossier

3. Assure-toi qu'ils sont au format `.mp3` et pas trop lourds (< 100KB)

## Suggestions de sons

- **work-complete.mp3** : Un son de succès/accomplissement (cloche, applaudissements légers)
- **break-complete.mp3** : Un son doux et apaisant (gong doux, notification subtile)
- **tick.mp3** : Un tic-tac d'horloge court (1 seconde max)
